Une application mobile a l&rsquo;avantage d&rsquo;être indépendante de la plateforme d&rsquo;exécution (Android, iPhone, iPad, BlackBerry, Palm,&#8230;)&nbsp; , étant donné que tout se fait sur le web, et à l&rsquo;aide de langages usuels que sont HTML, CSS & JavaScript.   * ### **<span style="font-family: book antiqua,palatino; font-size: 12pt;">JQuery Mobile</span>**

 

<p style="text-align: justify;">
  <span style="font-family: book antiqua,palatino; font-size: 12pt;"><a title="JQuery Mobile" href="http://jquerymobile.com/"><em><strong><img loading="lazy" class=" alignleft size-full wp-image-80" style="float: left;" src="https://charlesen.fr/wp-content/uploads/2011/01/images_jquery-logo.png" alt="Logo JQuery" title="Logo JQuery" width="235" height="61" />JQuery Mobile</strong></em></a> est un framework Javascript, optimisé pour les applications mobiles. On connaisait déjà JQuery, mais avec <a title="JQuery Mobile..." href="http://jquerymobile.com/"><em><strong>JQuery Mobile</strong></em></a>,&nbsp; le langage de script côté client prends encore plus d&rsquo;importance aux yeux des développeurs. On peut trouver sur le site de <a title="JQuery Mobile" href="http://jquerymobile.com/"><em><strong>JQuery Mobile</strong></em></a> une <a title="Documentation & Démo JQuery Mobile" href="http://jquerymobile.com/demos/1.0a2/">documentation</a> sous forme de démonstration vraiment très claire. Rien à dire là-desssus, ce Framework est vraiment très pratique, mais surtout très (trop) facile à prendre en main. Il devrait considérablement accélerer vos développements. Voir cette <a title="Introduction à jQuery Mobile, framework de développement mobile" href="index.php?option=com_content&view=article&id=93:introduction-a-jquery-mobile-framework-de-developpement-mobile&catid=35&Itemid=61">introduction au framework</a> et cet <a title="RSS Reader avec JQuery Mobile" href="http://net.tutsplus.com/tutorials/javascript-ajax/how-to-build-an-rss-reader-with-jquery-mobile-2/">exemple d&rsquo;utilisation</a> .<br /></span>
</p>

  * ### **<span style="font-family: book antiqua,palatino; font-size: 12pt;">PhoneGap</span>**

<p style="text-align: justify;">
  <span style="font-family: book antiqua,palatino; font-size: 12pt;"><a title="Site de PhoneGap" href="http://www.phonegap.com/"><em><strong><img loading="lazy" class=" alignleft size-full wp-image-81" style="float: left;" src="https://charlesen.fr/wp-content/uploads/2011/01/images_blog_pg_logo.png" alt="Logo de PhoneGap" title="Logo de PhoneGap" width="192" height="72" />PhoneGap</strong></em></a> est un Framework de développement de webapps. Il se sert du triplet HTML &#8211; CSS &#8211; JavaScript. Le Framework propose une <a title="API PhoneGap" href="http://docs.phonegap.com/">API</a> vraiment très riche, que je vous invite à découvrir<br /></span>
</p>

 

  * ### **<span style="font-family: book antiqua,palatino; font-size: 12pt;">Jo</span>**

<p style="text-align: justify;">
  <img loading="lazy" class=" alignleft size-full wp-image-82" style="float: left;" src="https://charlesen.fr/wp-content/uploads/2011/01/images_blog_jo_logo.png" alt="Logo Jo" title="Logo Jo" height="103" width="139" /><span style="font-family: book antiqua,palatino; font-size: 12pt;">&nbsp;</span><span style="font-size: 12pt;"><a title="Site du Fralework Jo" href="http://www.joapp.com/"><strong><em>Jo</em></strong></a> est un Framework HTML5 d&rsquo;application mobile très léger, compatible avec <strong><em>PhoneGap</em></strong>. <a title="Site du Framework Jo" href="http://www.joapp.com/"><strong><em>Jo </em></strong></a>se sert beaucoup plus de CSS pour gérer les évènements que de JavaScript, et est un donc beaucoup plus ouvert aux clients dont JavaScript est désactivé. La <a title="Documentation de Jo" href="http://www.joapp.com/docs/index.html">documentation du framework</a> est vraiment très riche et tient sur une page.<br /></span>
</p>

  * ### **<span style="font-size: 12pt;">Sencha Touch</span>**

<p style="text-align: justify;">
  <span style="font-size: 12pt;"><img loading="lazy" class=" alignleft size-full wp-image-83" style="float: left;" src="https://charlesen.fr/wp-content/uploads/2011/01/images_blog_sencha-logo.png" alt="Logo Sencha" title="Logo Sencha" height="120" width="120" srcset="https://charlesen.fr/wp-content/uploads/2011/01/images_blog_sencha-logo.png 250w, https://charlesen.fr/wp-content/uploads/2011/01/images_blog_sencha-logo-150x150.png 150w" sizes="(max-width: 120px) 100vw, 120px" /><a title="Sencha Touch" href="http://www.sencha.com/products/touch/"><em><strong>Sencha Touch</strong></em></a> est un Framework de développement d&rsquo;applications web riches. Il semble&nbsp; être plus pensé pour l&rsquo;IOS que pour les autres plateforme,s mais est un bon outil pour développer des applis pour Android ou tout autre client du même type.Les applications créées à l&rsquo;aide de ce framework semble être réellement natives. Là aussi la <a title="Voir la documentation de Sencha Touch..." href="http://dev.sencha.com/deploy/touch/docs/">documentation</a> est vraiment très riche.<br /></span>
</p>
